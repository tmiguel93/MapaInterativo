# Guia de uso

## Selecionar empresa

Use os botões no topo para alternar entre Matrix, Serena e Inner.

Ao trocar a empresa:

- o tema visual muda;
- os dados do mapa mudam;
- os filtros são limpos;
- o painel lateral carrega um estado disponível da empresa.

## Usar o mapa

No desktop, passe o mouse sobre um estado para ver o pop-in. Clique para fixar o estado no painel lateral.

No mobile, toque no estado para abrir os dados.

## Pesquisar

A busca aceita:

- nome do estado;
- UF;
- DDD;
- distribuidora.

Após pesquisar, o campo é limpo e a busca fica fixada em um chip. Para remover, clique no X do chip ou em Limpar filtros.

## Tabela operacional

No desktop, a tabela mostra todos os registros filtrados. Clique em uma linha para abrir o estado no painel lateral.

No mobile, a tabela vira uma lista de cards.

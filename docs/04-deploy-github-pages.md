# Deploy no GitHub Pages

Para publicar o projeto:

1. Acesse o repositório no GitHub.
2. Clique em Settings.
3. Entre em Pages.
4. Em Build and deployment, selecione Deploy from a branch.
5. Escolha branch `main`.
6. Escolha pasta `/root`.
7. Salve.

Depois de alguns minutos, o GitHub Pages irá gerar a URL pública.

Como o projeto é estático, não há necessidade de build, servidor ou banco de dados.
